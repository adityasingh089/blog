Cryptography

Cryptography is the study and practice of techniques for secure communication in the presence of third parties. It involves creating and analyzing protocols that prevent adversaries from gaining access to private information.

Key Concepts:

  • Confidentiality: Ensures only authorized parties can access information.
  • Integrity: Verifies data hasn't been tampered with.
  • Authentication: Verifies the identity of communicating parties.
  • Non-repudiation: Prevents denying the origin of a message.
  • Key Management: Securely manages cryptographic keys.

Types of Cryptography:

  • Symmetric Cryptography: Uses a single key for encryption and decryption (e.g., AES).
  • Asymmetric Cryptography: Uses a pair of keys for encryption and decryption (e.g., RSA).
  • Hash Functions: Ensure data integrity and create digital signatures (e.g., SHA-256).

Applications:

  • Secure Communication: Used in messaging apps, email encryption, VPNs, and secure web browsing (HTTPS).
  • Data Protection: Secures data at rest and data in transit.
  • Authentication and Access Control: Plays a role in user authentication and access control mechanisms.
